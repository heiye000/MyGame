### 记录一些查询的信息
1.GUIDE Trigger方式
| Trigger            | 作用         | ARPG 推荐度 | 适合什么         |
| ------------------ | ---------- | -------- | ------------ |
| **Pressed**        | 按下瞬间触发一次   | ⭐⭐⭐⭐⭐    | 攻击、翻滚、技能、交互  |
| **Released**       | 松开时触发      | ⭐⭐       | 蓄力结束、弓箭发射    |
| **Hold**           | 按住达到一定时间   | ⭐⭐⭐⭐     | 蓄力攻击、长按奔跑    |
| **Tap**            | 快速点击       | ⭐⭐⭐      | Dash、双模式技能   |
| **Pulse**          | 按住期间周期触发   | ⭐⭐       | 连续射击、机关      |
| **Combo**          | 特定按键序列     | ⭐⭐       | 格斗游戏连招       |
| **Chorded Action** | 组合键        | ⭐⭐⭐      | Shift+A、LT+X |
| **Hair**           | 输入阈值（模拟摇杆） | ⭐        | 手柄           |
| **Stability**      | 输入稳定检测     | ⭐        | 手柄、触摸        |
